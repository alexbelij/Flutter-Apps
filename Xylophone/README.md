# Xylophone

Playing Xylophone Sounds with this App.

## What I learned Building this App

- Learned how to add and remove public [Flutter packages](https://pub.dev/flutter/packages) in App.
- How to play audio files in local library.
- **Packages Used in this App**

    - [audioplayers 0.15.1](https://pub.dev/packages/audioplayers)
